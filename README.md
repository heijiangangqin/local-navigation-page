# 🏠 自定义浏览器首页

一个美观、功能丰富的浏览器默认首页，模仿现代桌面界面设计，包含时间显示、搜索栏、小部件和可管理的应用卡片。

## ✨ 主要功能

### 🕐 时间显示
- 实时显示当前时间（时:分:秒）
- 显示当前日期和农历信息
- 自动更新，无需刷新页面

### 🔍 搜索功能
- 集成百度搜索
- 支持回车键快速搜索
- 简洁的搜索界面
- 实时天气信息显示

### 📅 小部件系统
- **天气小部件**: 实时显示当前天气信息，包含温度、湿度、风向等
- **日历小部件**: 显示当前月份日历，高亮当天日期
- **记事本小部件**: 支持实时编辑，自动保存到本地存储
- **待办事项小部件**: 完整的待办管理功能，支持增删改查、优先级设置、完成状态跟踪

### 🎯 卡片管理
- **添加卡片**: 支持自定义名称、链接地址和图标
- **编辑卡片**: 修改现有卡片的信息
- **删除卡片**: 移除不需要的卡片
- **图标预览**: 实时预览卡片图标
- **数据持久化**: 自动保存到浏览器本地存储

### 🎨 界面设计
- 渐变背景，视觉效果丰富
- 毛玻璃效果，现代感十足
- 响应式设计，支持各种屏幕尺寸
- 平滑动画效果，用户体验优秀
- 天气小部件蓝色渐变主题

## 🚀 快速开始

### 1. 下载项目
```bash
git clone [项目地址]
cd homepage
```

### 2. 打开首页
直接在浏览器中打开 `index.html` 文件即可使用。

### 3. 测试拖拽功能
- 打开 `test-drag.html` 文件可以单独测试卡片拖拽排序功能
- 打开 `test-widget-drag.html` 文件可以单独测试小部件拖拽排序功能

### 4. 设置为浏览器首页

#### Chrome/Edge 浏览器
1. 打开浏览器设置
2. 找到"启动时"选项
3. 选择"打开特定页面或一组页面"
4. 点击"添加新页面"
5. 输入 `index.html` 的完整路径
6. 保存设置

#### Firefox 浏览器
1. 打开浏览器设置
2. 找到"主页"选项
3. 在"主页和新窗口"中输入 `index.html` 的完整路径
4. 保存设置

## 📱 使用方法

### 页面布局
- **顶部**: 时间显示栏
- **搜索栏**: 居中搜索框
- **应用区域**: "我的应用"卡片区域（上方）
- **小部件区域**: 天气、日历、记事本、待办事项（下方）

### 添加新卡片
1. 点击"添加卡片"按钮
2. 填写卡片名称（必填）
3. 填写链接地址（必填）
4. 填写图标URL（可选）
5. 点击"保存"按钮

### 编辑卡片
1. 将鼠标悬停在卡片上
2. 点击编辑按钮（✏️）
3. 修改卡片信息
4. 点击"更新"按钮

### 删除卡片
1. 将鼠标悬停在卡片上
2. 点击删除按钮（🗑️）
3. 确认删除

### 拖拽排序卡片
1. 将鼠标悬停在卡片上，显示拖拽手柄（⋮⋮）
2. 点击并拖拽手柄或直接拖拽卡片
3. 将卡片拖拽到目标位置
4. 释放鼠标完成排序

### 拖拽排序小部件
1. 将鼠标悬停在小部件上，显示拖拽手柄（⋮⋮）
2. 点击并拖拽手柄或直接拖拽小部件
3. 将小部件拖拽到目标位置
4. 释放鼠标完成排序
5. 支持天气、日历、记事本、待办事项的重新排序

### 使用记事本
- 直接在记事本小部件中输入内容
- 内容会自动保存到本地存储
- 刷新页面后内容依然保留

### 天气功能
- 自动获取当前城市天气信息
- 显示温度、湿度、风向、风力等详细信息
- 每30分钟自动更新天气数据
- 支持多种天气状况的图标显示

### 待办功能
- 直接输入添加待办事项（无需弹窗）
- 支持回车键快速添加
- 勾选完成状态
- 一键删除待办
- 显示完成进度和统计信息
- 按优先级和完成状态排序
- 数据自动保存到本地存储

### 搜索功能
- 在搜索栏输入关键词
- 按回车键或点击搜索按钮
- 将在新标签页中打开百度搜索结果
- 简洁的搜索界面，专注搜索功能

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + N` | 添加新卡片 |
| `Enter` | 在待办输入框中按回车添加待办 |
| `Escape` | 关闭模态框 |

## 🔧 高级功能

### 导出卡片数据
在浏览器开发者工具控制台中执行：
```javascript
exportCards()
```
这将下载一个包含所有卡片数据的JSON文件。

### 导入卡片数据
在浏览器开发者工具控制台中执行：
```javascript
// 假设你有一个文件对象 file
importCards(file)
```

### 自定义样式
可以通过修改 `styles.css` 文件来自定义界面样式：
- 修改背景渐变
- 调整卡片大小和布局
- 更改颜色主题
- 自定义动画效果
